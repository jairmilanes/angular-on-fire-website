Starting a software project takes time, deciding which tools to use, configuring CI/CD pipelines, maintaining code quality, protecting code against unstable changes, unit tests, e2e tests, in fact there is a too much to think about.

Angular On Fire was created to take a bit of that burden off your shoulders.

Angular is one of the most powerful front-end frameworks in the market, coupled with Angular-Cli it is like Hulk on gamma steroids.

Angular On Fire harnesses that power by giving you a Angular Universal (SSR) starter project, packet with the latest RXJS & Flex Layout, to make creating fast, responsive, SEO friendly applications a breeze

Firebase has a lot to offer, it's free to start and has a pay only for what you use model, that makes it perfect for start up's and independent developers that want to get powerful apps out, without worrying about infrastructure.

Out of the box Angular On Fire provides:

  • Multi environment Firebase Hosting deployment (production & staging)
  • Firebase services integration with AngularFire 2

There is always room for improvement, and we are always accepting pull requests bug fixes and improvements.

All you have to do is submit a PR following the guidelines in the template and we will take a look as son as we can.
